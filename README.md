# PromptQuest Landing Page Theme

Custom WordPress theme for PromptQuest landing page.

## Installation

1. Upload theme to WordPress
2. Activate theme
3. Create new page
4. Select template: "PromptQuest Landing Page (Full Width)"
5. Paste landing page HTML in content
6. Publish

## Auto-Deploy Setup

This theme is connected to GitHub and auto-deploys via WP Pusher.

- Repository: github.com/shikha-work/promptquest-theme
- Auto-deploy: Enabled (push to main branch)

## Files

- `style.css` - Theme header info
- `functions.php` - WordPress functions
- `index.php` - Fallback template
- `page-landing.php` - Landing page template
- `README.md` - This file
